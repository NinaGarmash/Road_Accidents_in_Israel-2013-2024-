# Phase 3b — E-Scooter Accidents Tel Aviv: CBS 2013–Dec 2025

## What changed from Phase 3a

| | Phase 3a | Phase 3b |
|---|---|---|
| **Data cut** | CBS 2013–Jun 2025 | CBS 2013–Dec 2025 (full year) |
| **Total accidents** | Up to mid-2025 | Full 2025 calendar year |
| **Model tooling** | Manual Power BI Desktop | Updated via [MCP Power BI Modeling Server](https://github.com/jonathanscholtes/PowerBI-MCP-Server) |

## Why Phase 3b exists

Phase 3a was submitted to the ISTRC Conference (Technion, January 2025) using a mid-year data cut.
After the full 2025 CBS microdata was released, the dataset was extended to cover the complete calendar year.

The data model was also updated using an **MCP (Model Context Protocol) Power BI server**, which enables
programmatic DAX and schema changes directly from an AI assistant — replacing manual edits in Power BI Desktop.

Full change log: [Google Docs — Phase 3b Model Changes](https://docs.google.com/document/d/1H_1aDwIcxfwuQ1YMFJ3R9Ie5GDlaWYs4imnv3xnI8UQ/edit?tab=t.0)
